# contact-form

A small React + TypeScript project containing a contact form.

This repository is intentionally shipped with a defect that causes the app to behave badly during local development. Your job is to identify and fix the issue, then add a unit test to ensure it stays fixed.

## Requirements

- Node.js 20+
- npm 10+ (or compatible)

## Getting started

Install dependencies:

```bash
npm install
```

Run the app:

```
npm run dev
```

Run tests:

```
npm run test
```
